# Model-Driven Application

This project is a business application built using Microsoft Power Apps (Model-Driven App).

The application demonstrates how to design and build a complete data-driven system using Dataverse.

## Features

- Custom tables and relationships
- Forms and views design
- Business Process Flow implementation
- Data management 

## Technologies

- Microsoft Power Apps (Model-Driven)
- Dataverse
